# 微信小程序动态定价系统

这是一个用于个人开发验证的微信小程序动态定价 MVP。当前版本为纯前端单页应用，不依赖云函数，适合快速调试商品定价策略。

## 当前功能

- 根据成本、基准价、竞品价、目标毛利率生成价格区间
- 根据需求、库存、渠道、时段、用户身份计算建议售价
- 展示利润、利润率、预估收入、因子贡献、策略解释和风险提示
- 支持晚高峰、清库存、节假日三个快速场景
- 支持保存最近方案、清空记录、复制可读定价摘要

## 模块结构

```text
miniprogram/pages/index/
  index.js       页面交互层，只负责表单、标签页和按钮事件
  index.wxml     页面结构
  index.wxss     Google/Material 风格样式

miniprogram/utils/
  pricingEngine.js    独立定价计算模块
  pricingStorage.js   独立本地存储模块
```

## 说明

原先“策略数据”显示为 JSON，是为了开发阶段复制到数据库或接口调试。现在页面已改为业务摘要和因子说明，复制按钮输出可读方案文本，不再在界面展示代码块。
