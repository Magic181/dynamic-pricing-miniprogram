# 微信小程序动态定价系统

这是一个用于个人开发和验证的动态定价小程序雏形。当前版本不依赖后端，打开微信开发者工具即可在首页体验定价计算。

## 功能

- 根据成本、基准价、竞品价、目标毛利率计算价格保护区间
- 根据需求热度、库存水平、销售渠道、时段场景、用户身份动态调整建议价
- 展示单件利润、利润率、预估收入和策略解释
- 支持晚高峰、清库存、节假日三种快速场景
- 可复制策略 JSON，后续可接入云数据库或自己的服务端

## 项目结构

```text
miniprogram/pages/index/
  index.js     定价计算与交互逻辑
  index.wxml   动态定价控制台页面
  index.wxss   页面样式
```

## 后续可扩展

- 接入云开发数据库保存商品、价格策略和调价历史
- 增加订单转化率、天气、节假日、活动券等外部因子
- 增加管理员登录和商品列表
- 将定价逻辑迁移到云函数，避免前端暴露核心规则
